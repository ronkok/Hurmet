Drawings

Hurmet’s draw environment enables function plotting. For instance, to get a sine wave, one can write the following code into a math zone:

draw()
    title "sin x"
    frame 250, 150
    view -5, 5, -3
    axes 2, 1, "labels"
    strokewidth 2
    plot sin(x), 51
    text [1.6, 1.35], "sin x"
end
